Nossos Valores



























Transparência

O oculto é um ato o qual não toleramos, pois tudo o que é justo é feito Transparentemente, mas o que é malicioso é feito em secreto; ou é mascarado de verdade.



























Respeito

O respeito é o pilar precípuo de toda boa relação, logo, se uma empresa perde sua base, como subsistirá? Se há desrespeito entre nós mesmos, ou para com nossos clientes, como subsistiremos? Já não o seremos nós.



























União

Como o ferro forja o ferro, nós nos forjamos a nós mesmos, uns para com os outros. Todos dependem de um, e um, de todos.







Nossos Serviços



























MySQL Server

Utilizando do MySQL Server (Workbench), tanto os dados de nossos clientes quanto o nível de gás obtido pela detecção do sensor MQ-2 ficam armazenados em nosso banco de dados.



























API

Através da API, fazemos uma aplicação para que, com os dados capturados pelo sensor MQ-2, por meio do banco de dados (MySQL Server), os valores, em forma de Dashboard, ,possam ser visualizados.



























Sensor de Gás (MQ-2 )

Com o sensor MQ-2, nós fazemos a detecção precisa e instantânea dos gáses presentes na área, garantindo segurança e sapiência quanto à situação da região em questão.
